abstract: |-
  We propose **Fun-CineForge**, a unified solution for zero-shot movie dubbing that combines an end-to-end pipeline for large-scale multimodal dubbing data with a multimodal model purpose-built for diverse cinematic scenarios. From this pipeline we construct **CineDub**, the first dedicated movie-dubbing dataset, released open-source. Across monologue, narration, dialogue, and complex multi-speaker scenes, Fun-CineForge surpasses SOTA in audio quality, temporal alignment, lip-sync, timbre similarity, and instruction following — and is the first dubbing model to align long videos via the temporal modality. The model has been open-sourced through Alibaba Tongyi's official speech repository (10 K+ downloads), with demos on HuggingFace and ModelScope.
